# Sistema de Gestión de Datos en Python
Proyecto para gestionar datos con Python.  
Permite registrar, validar y manipular información de manera estructurada.
## Estructura
SistemaGestionDatos/
├── main.py
├── README.md
├── .gitignore
└── modulos/
    ├── datos_basicos.py
    ├── validaciones.py
    ├── menu.py
    ├── gestion_datos.py
    └── funciones_utiles.py
## Tecnologías utilizadas
- Python 3.10 Lenguaje principal para el desarrollo del sistema.
- Visual Studio Code → Editor de código para escribir y organizar el proyecto.
- Git & GitHub → Control de versiones y repositorio remoto.
- Markdown → Documentación en el archivo README.md.
## Ejecución
Clonar el repositorio y ejecutar el archivo principal:
en bash:
git clone https://github.com/Kattasaky/SistemaGestionDatos.git
cd SistemaGestionDatos
python main.py
